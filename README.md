# Canadian Hospital Readmissions Analysis

Analyzing 30-day hospital readmission rates across Canadian provinces using CIHI data (2020-2025).

## Data Source
Canadian Institute for Health Information (CIHI)

## Tools
- SQL (DB Browser for SQLite)

## Questions Being Explored
- Which provinces have the highest readmission rates?
- Are rates improving or worsening over time?
- How do BC hospitals compare nationally?
